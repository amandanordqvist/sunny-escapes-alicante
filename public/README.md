# Public Assets Directory

This directory contains public assets for the Sunny Escapes Alicante website.

## Required Files

### Hero Video
Add a video file named `hero-video.mp4` to this directory. The video should:
- Be in MP4 format
- Show beautiful scenery of Alicante or luxury properties
- Be around 30 seconds long and loop seamlessly
- Be optimized for web (compressed to reduce file size while maintaining quality)
- Not contain any audio (it will be muted)

Example video content:
- Aerial shots of coastal properties
- Sunset views of the Mediterranean
- Luxury villa exteriors and pools
- Beautiful Alicante landscapes

You can find suitable stock videos from websites like:
- Pexels Videos
- Shutterstock
- iStock
- Adobe Stock
